Tablatures Manager

Pourquoi?

Tablatures Manager se veut un petit logiciel pratique. Son utilité réside dans la simplification et la souplesse de la gestion de vos fichiers de tablatures.

Comment?

Tablatures Manager liste un répertoire de votre choix et y recherche tous les fichiers dits "Guitar pro", ensuite il vous proprose une gestion plus efficace que l'utilisation de répertoire et sous répertoire pour classer vos fichiers.

Utilisation

Lorsque vous démarrerez Tablatures Manager pour la premiére fois, l'application vous demandera quel répertoire elle doit lister à la recherche de vos fichiers dits "Guitar pro". Ensuite vous tomberez sur la fenêtre principale de l'application, que vous rencontrerez systématiquement aux démarrages suivants de l'application. Cette fenêtre est la fenêtre de recherche. Elle permet d'appliquer un seul et unique filtre à la recherche d'un fichier GP. Et pour ceci elle s'aide des informations que l'utilisateur à déjà renseigné. Aisni, par exemple, si vous sélectionnez un auteur particulié dans la liste déroulante titrée Auteur, seuls les fichiers dont l'auteur à été renseigné et qui correspondent à l'auteur recherché seront présent dans la liste déroulabte de fichiers.

Les menus et les différents boutons

  • Le bouton Valider permet de lancer le fichier GP dans les fenêtre de recherche et d'affichage. Et de valider la modification dans la fenêtre de modification.
  • Les boutons Annuler et Quitter permetent dans le premier cas de quitter la fenêtre en cours et dans le second cas de fermer l'application.
  • Le choix Modifier dans le premier menu permet de modifier la fiche du fichier GP actuel sélectionné dans la liste déroulante des fichiers.
  • Le choixAfficher dans le premier menu permet d'afficher la fiche du fichier GP actuel sélectionné dans la liste déroulante des fichiers.
  • Le choix Voir le xml lance l'affichage du fichier qui permet la sauvegarde des données avec le navigateur par défaut.
  • Le choix Aide permet l'affichage de cette mê fenêtre avec le navigateur par défaut.
  • Mettre à jour le xml permet la mise à jour du fichier de données xml sans redémarrage de l'application.
  • Remise à zéro de la recherche permet de remettre à zéro toutes les listes déroulante de la fenêtre de recherche.

Comment ajouter de nouveaux fichiers ou en supprimer?

Ne vous en faite pas pour ça, à chaque démarrage de l'application la différence entre les fichiers présent dans le répertoire renseigné, et ses sous-répertoire, puis les chemins des fichiers sauvegardés comme tels dans le fichier xml de donné génère une action de l'application, qui selon le cas ajoute ou supprime dans le fichier de données xml, le chemin du fichier correspondant.

Si vous êtes en pleine utilisation de l'application, pour ne pas à redémarrer l'application une option dans le menu vous permez de produire cette action de recherche de différence et de modification du fichier de donné en conséquence.
